# Routing:
- We know all the HTTP methods Describes the INTENT of the Request, what do we want to do with the Resource in the Server.
- The role of ROUTING is, "WHERE" do you want to send your "INTENTION", OR "WHICH" Resource on which you want to do hte intent on.
- Routing can be Defined as MAPPING URL PARAMETERS to a Server side logic.
- The METHOD & ROUTE together maps to a particular Server Logic.
