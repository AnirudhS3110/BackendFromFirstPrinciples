# HTTP Headers:
- It consists of metadata of the Recepient
- It acts as Remote Control on the Server Side, it allows the Client to Send Instruction to the server influencing how the server responds or Processes Requests.
- Servers can use ```Cache Control``` to fix the Duration of the Cache in the client.
### Types of HTTP Headers:
#### Request Headers:
- 1) User- Agent - tells which type of Client is that, whether its Browser or Postman or any other server
- 2) Authorization - Carries the **Bearer Tokens** to the Server
- 3) Cookie
- 4) Accept - provides info of what type of content we are accepting , whether it is JSON or HTML files
#### General Headers:
- Data
- CAche controkl
- Connection

#### Reperesentation Headers:
- Explains the Representation of the Information/resource which is being Transmitted
- Provides info about Body/Headers
- - COntent type
- - Content Length
- - Content Embedding
- - ETag
