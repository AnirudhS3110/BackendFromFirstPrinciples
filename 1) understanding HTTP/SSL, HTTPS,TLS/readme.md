## SSL:
- Protocol to securing Communication between the CLient and the Server, by encrypting the data flowing between client and server, so that attackers can do nothing.
- Encryption machanism between Clients and Server.
- Now Outdated due to Security vulnerrabilities, now Replaced by TLS.

## TLS:
- Modern and Most Secure Version of SSL.

## HTTPS:
- Its HTTP, but its more secure.
- Security provided by TLS.
- HTTP uses TLS
- This TLS encypts the CONNECTION between the Client and the Server.
