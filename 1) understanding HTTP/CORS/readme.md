# CORS 
- Same Origin Policy 
- Browsers use Same Origin Policy to restrict webpages making request from DOMAIN different from their webpage. It allows request between same domain/subdomain.
- Its a Security Mechanism implemented in Browsers, to control how Web Apps interacts with Resources hosted on Different Domains!
- It ensures that Resources can be accessed by webpages only inside their own Domain. 
- Usually the JS codes which the Browser runs is ran in Isolated Environment or Sandbox Environment.
