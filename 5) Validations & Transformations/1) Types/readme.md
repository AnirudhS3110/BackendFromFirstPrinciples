- It Checks the value with the Preferred Regex. example: Phone number, email
## 2) Semantic
- For example the user shd not enter a date in Future in Date of Birth Field.
- Age example, its not possible to have an age of 35.

## 3) Type Validation
- Checks the Data type of the value.
